# 散布計画 計算 (spray_calc)

やまびこ(共立)散布機（ブームスプレーヤ／スピードスプレーヤ／動噴 等）向けの
散布量シミュレータ。スマホ対応の静的 Web アプリ（Vite + React + TypeScript）です。

## 機能

1. **バランス計算** — 反当散布量 R・散布幅 W・走行速度 V・ノズル総吐出量 Q・
   圃場係数 F のうち 1 項目を空欄にすると、残り 4 項目から自動で逆算します。
   基本式は `Q = R × W × V × F ÷ 60`。
   入力検証・計算式表示・結果コピー・URL 共有・入力値保存（W/V/F）に対応。
2. **ノズル設定から計算** — ノズル（モデル or 手入力）と運転圧力・本数から
   ノズル総吐出量 Q を算出（流量は圧力の平方根に比例: `q ∝ √P`）。結果は
   バランス計算へ反映できます。
3. **機器・ノズル提案** — 作物・散布条件から必要総吐出量を求め、やまびこ(共立)の
   適合機種（ブーム／スピードスプレーヤ／ラジコン動噴／セット動噴／乗用管理機）と
   ノズル設定を提案します。作物→散布方法→機種/ノズルの逆算と、機種×作物×ノズルの
   適合マトリクスに対応。動噴・SSV は **吸水量 × 0.8** を実用吐出量として判定します。

## データスキーマと出典

データは `src/data/` にあり、各レコードは `source`（`provenance` / `verified` /
`sourceUrl` / `updatedAt`）を持ちます。機種は `machines.ts`、ノズルは `nozzles.ts`、
作物は `crops.ts`、散布方法は `sprayMethods.ts`、機種×作物×ノズルの適合は
`compatibility.ts`。実用吐出量（吸水量×0.8 等）は `lib/selection.ts`。

- `iso-standard` … ISO 10625 規格に基づく値（平板扇形ノズルの色分け・流量）
- `maker-catalog` … やまびこ(共立)公式の製品情報。公式詳細ページで確認できた機種
  （BSM511R/512R/513B/E、SSV6150F、WGR617V-10、VRC4570F2-8 等）を `verified:true`、
  相互確認のみのものは `verified:false`（要確認）で収録
- `estimated` … 未確認の推定値（作物×散布方法×ノズルの慣行マッピング等）

> やまびこ製品に限定。ノズルは純正＋ISO標準を主体とし、他社（ヤマホ）は技術参考
> リンクのみ残しています。乗用管理機など諸元未確認の機種は順次拡充します。

まず公開情報で初期化し、後からメーカ公式の実データに差し替えれば、UI のバッジ
（確認済／要確認）と提案精度がそのまま向上します。`verified` を `true` にする際は
一次情報での確認を前提とします。

> 提案結果・散布量の目安は参考値です。実際の散布は機器の取扱説明書・農薬ラベル・
> 地域の防除基準に従ってください。
