# 散布計画 計算 (spray_calc)

ブームスプレーヤ／スピードスプレーヤ向けの散布量シミュレータ。スマホ対応の
静的 Web アプリ（Vite + React + TypeScript）です。

## 機能

1. **バランス計算** — 反当散布量 R・散布幅 W・走行速度 V・ノズル総吐出量 Q・
   圃場係数 F のうち 1 項目を空欄にすると、残り 4 項目から自動で逆算します。
   基本式は `Q = R × W × V × F ÷ 60`。
   入力検証・計算式表示・結果コピー・URL 共有・入力値保存（W/V/F）に対応。
2. **ノズル設定から計算** — ノズル（モデル or 手入力）と運転圧力・本数から
   ノズル総吐出量 Q を算出（流量は圧力の平方根に比例: `q ∝ √P`）。結果は
   バランス計算へ反映できます。
3. **機器・ノズル提案** — 作物・散布条件から必要総吐出量を求め、適合する
   散布機・ノズル（運転圧力・本数）を必要吐出量ベースで提案します。

## データスキーマと出典

ノズル・散布機のデータは `src/data/` にあり、各レコードは `source`
（`provenance` / `verified` / `sourceUrl` / `updatedAt`）を持ちます。

- `iso-standard` … ISO 10625 規格に基づく値（平板扇形ノズルの色分け・流量）
- `maker-catalog` … メーカ公式の製品情報（共立／やまびこ ブームスプレーヤ等）
- `estimated` … 未確認の推定値

まず公開情報で初期化し、後からメーカ公式の実データに差し替えれば、UI のバッジ
（確認済／要確認）と提案精度がそのまま向上します。`verified` を `true` にする際は
一次情報での確認を前提とします。

> 提案結果・散布量の目安は参考値です。実際の散布は機器の取扱説明書・農薬ラベル・
> 地域の防除基準に従ってください。
